Discussion

Asymmetry detection Performance

Quantitative EEG analysis provides continuous and objective indicators of anesthetic effects and could be used to decide when to perform memory recall and language tests (Tu 2015). This is task is usually performed by a clinical
neuropsychologist. Therefore, in this study, the annotations provided by the clinical neuropsychologist are used as the golden standard. Hence, quantitative EEG based asymmetry detection is optimized with respect to the clinical neuropsychologist's annotations. Our optimization is based on two quantitative EEG measures used for asymmetry detection during Wada test the Brain Symmetry Index (BSI) (Peters 2012) and relative Filterbank Bank Energy (rFBE) in the 0.7-2.3 Hz frequency range (Tu 2015). First, we evaluated which electrodes, when used to calculate BSI and rFBE, resulted in optimal asymmetry detection performance. In case of BSI, the electrode pair F3-F4 resulted in the highest performance of AUC = 0.96 (median value of 53 patients). Adding additional electrodes did not result in significant improvements. For rFBE, highest performance of AUC = 0.96 was achieved using either electrode pair C3-C4 or the midline electrode Cz. Second, the combination of BSI and rFBE using linear discriminant analysis revealed that by simply taking the average of both QEEG measures results in a relative performance increase of 50% (AUC = 0.98, median value of 53 patients).

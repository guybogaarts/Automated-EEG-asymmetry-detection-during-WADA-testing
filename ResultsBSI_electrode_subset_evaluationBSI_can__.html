BSI electrode subset evaluation

BSI can be calculated using any combination of one or more left-right electrode pairs. With 8 left-right electrode pairs this yields a total of 255 combinations. To investigate the importance of a single electrode pair, all possible combinations including or excluding a particular pair are compared. (Het stuk hiervoor hoort volgens mij nog in de methode thuis) Figure 1 shows overall AUC value distributions for BSI for all possible electrode pair combinations divided in two groups. The left boxplot is the group where the mentioned electrode pair is excluded and the right boxplot where this electrode pair is included. The central blue line indicates the group average? AUC value for just this single electrode pair. Clearly electrode pair F3-F4 has the highest single pair AUC as well as the highest AUC values for the group where this pair is included. When this electrode pair is not used in the BSI calculation, performance is significantly lower, irrespective of the other included electrode pairs (p<0.0001, Wilcoxon ranked sum test). Only 3 other electrode pair combinations yield a slightly higher performance (F3-F4 + P3-P4, or C3-C4, or C3-C4 and O1-O2 dit is niet duidelijk genoeg). However, when the AUC is calculated per patient, the Wilcoxon signed rank test indicates that these differences are not statistically significant (p = 0.52, p=0.57 and p=0.85). The single electrode pair AUC for F3-F4 is statistically significantly different from all other 7 single electrode pairs. Figure 2A shows the AUC values for the BSI calculated using the single left-right electrode pairs. Each boxplot indicates the distribution of the AUC values for the 53 patients for each left-right electrode pair. BSI based on electrode pair F3-F4 achieves the highest median AUC value of 0.96 (0.89 and 0.99, 25th and 75th percentile).
